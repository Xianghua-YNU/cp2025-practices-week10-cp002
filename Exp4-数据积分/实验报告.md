## 二、实验原理

### 2.1 梯形积分法

对于离散的速度数据点$(t_0,v_0),(t_1,v_1),...,(t_N,v_N)$，总距离（位移）可通过梯形法则计算：

$$
\text{距离} \approx \sum_{i=0}^{N-1} \frac{v_i + v_{i+1}}{2} (t_{i+1} - t_i)
$$

### 2.2 累积距离计算

使用`scipy.integrate.cumulative_trapezoid`计算每个时间点的累积距离：

$$
\text{累积距离}(t_i) = \int_{t_0}^{t_i} v(t)dt
$$

```
import numpy as np
from scipy.integrate import cumulative_trapezoid
import matplotlib.pyplot as plt
import os

def main():
    try:
        # 1. 获取数据文件路径
        data_file = r"C:\Users\84786\Desktop\velocities.txt"

        # 2. 读取数据
        data = np.loadtxt(data_file)
        t = data[:, 0]  # 时间列
        v = data[:, 1]  # 速度列

        # 3. 计算总距离
        total_distance = np.trapz(v, t)
        print(f"总运行距离: {total_distance:.2f} 米")

        # 4. 计算累积距离
        distance = cumulative_trapezoid(v, t, initial=0)

        # 5. 绘制图表
        plt.figure(figsize=(10, 6))
        plt.plot(t, v, 'b-', label='Velocity (m/s)')
        plt.plot(t, distance, 'r--', label='Distance (m)')
        plt.title('Velocity and Distance vs Time')
        plt.xlabel('Time (s)')
        plt.ylabel('Velocity (m/s) / Distance (m)')
        plt.legend()
        plt.grid(True)
        plt.show()

    except FileNotFoundError:
        print("错误：找不到数据文件")
        print("请确保数据文件存在于正确路径")

if __name__ == '__main__':
    main()
```

### 4.2 误差分析
分析梯形积分法计算距离的误差来源
- 1.采样间隔

梯形积分法是基于离散的数据点进行计算的，它假设在相邻两个采样点之间速度是线性变化的。当采样间隔较大时，实际的速度变化可能并非线性，从而导致较大的误差。例如，如果速度在短时间内有剧烈的波动，但采样间隔没有足够小到捕捉这些变化，那么梯形积分法所近似的速度变化就会与实际情况有偏差，进而影响距离的计算结果。
- 2.数据噪声

实验获取的速度数据可能包含噪声，这些噪声会使速度数据偏离真实值。在梯形积分过程中，噪声会被累积，导致最终计算出的距离与真实距离存在误差。例如，随机的测量误差可能会使某些速度数据点出现异常值，这些异常值在积分过程中会对结果产生较大的影响。
- 3.函数逼近

梯形积分法本质上是用梯形来逼近速度 - 时间曲线下的面积。对于一些复杂的速度函数，梯形的近似可能不够精确。如果速度函数具有高阶导数或非平滑的特性，梯形积分法可能无法准确地描述其变化，从而引入误差。

- 4.为了减小误差，可以采取以下措施：

(1) 减小采样间隔，以更准确地捕捉速度的变化。

(2) 对数据进行滤波处理，去除噪声的影响。

(3) 使用更高级的数值积分方法，如辛普森法则等，以提高积分的精度。
