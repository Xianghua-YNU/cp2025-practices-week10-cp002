## 二、实验原理

### 2.1 Simpson 法则

简述 Simpson 法则的积分思想、公式推导及适用条件（ $N$ 必须为偶数），并给出公式：
 $\int_a^b f(x)dx \approx \frac{h}{3}\left[f(x_0) + 4\sum_{i=1,3,5,...}^{N-1}f(x_i) + 2\sum_{i=2,4,6,...}^{N-2}f(x_i) + f(x_N)\right]$


1. **积分思想**  
   Simpson 法则是一种数值积分方法，它通过用二次多项式（抛物线）来近似被积函数，从而提高积分精度。具体来说，Simpson 法则将积分区间分为多个等长的小区间，在每个小区间内用二次插值多项式来逼近函数值，然后对这些多项式的积分求和。

2. **公式推导**  
   - 将积分区间 $[a, b]$ 分为 $N$ 个等长小区间，步长为 $h = \frac{b-a}{N}$，其中 $N$ 必须为偶数。
   - 在每个小区间上，用二次插值多项式 $P(x)$ 逼近函数 $f(x)$，即：
      $P(x) = f(x_0) + \frac{x-x_0}{h}[f(x_1)-f(x_0)] + \frac{(x-x_0)(x-x_1)}{2h^2}[f(x_2)-2f(x_1)+f(x_0)]$
   - 对 $P(x)$ 在每个小区间上积分，得到近似值。
   - 将所有小区间的积分结果相加，得到 Simpson 法则的公式：
     $\int_a^b f(x)dx \approx \frac{h}{3}\left[f(x_0) + 4\sum_{i=1,3,5,...}^{N-1}f(x_i) + 2\sum_{i=2,4,6,...}^{N-2}f(x_i) + f(x_N)\right]$

3. **适用条件**  
   - $N$ 必须为偶数，因为 Simpson 法则需要每两个小区间构成一个抛物线。
   - 被积函数 $f(x)$ 在区间 $[a, b]$ 上必须连续，且具有较好的光滑性（即导数不发生剧烈变化）。

4. **Simpson 法则公式**  
   Simpson 法则的公式为：
   $\int_a^b f(x)dx \approx \frac{h}{3}\left[f(x_0) + 4\sum_{i=1,3,5,...}^{N-1}f(x_i) + 2\sum_{i=2,4,6,...}^{N-2}f(x_i) + f(x_N)\right]$
   其中：
   - $x_0 = a, x_N = b$，其余点为 $x_i = a + i \cdot h$。
   - $h = \frac{b-a}{N}$ 是步长。

Simpson 法则相比梯形法则具有更高的精度，尤其在被积函数是二次多项式或接近二次多项式时，能够给出精确结果。

### 2.2 梯形法则

简述梯形法则的积分思想、公式推导，并给出公式：
 $\int_a^b f(x)dx \approx h\left[\frac{f(x_0)}{2} + f(x_1) + \cdots + f(x_{N-1}) + \frac{f(x_N)}{2}\right]$

1. **积分思想**  
   梯形法则是一种数值积分方法，它通过将被积函数在每个小区间上用线性函数（直线）近似，从而将积分区域分割成多个梯形的面积之和。梯形法则的核心思想是用梯形的面积来近似曲线下的面积。

2. **公式推导**  
   - 将积分区间 $[a, b]$ 分为 $N$ 个等长小区间，步长为 $h = \frac{b-a}{N}$。
   - 在每个小区间 $[x_i, x_{i+1}]$ 上，用直线连接 $f(x_i)$ 和 $f(x_{i+1})$，形成一个梯形。
   - 梯形的面积为：
     $\text{面积} = \frac{1}{2} \cdot \text{底边长} \cdot (\text{左高} + \text{右高}) = \frac{h}{2} \cdot [f(x_i) + f(x_{i+1})]$
   - 将所有梯形的面积相加，得到梯形法则的公式：
     $\int_a^b f(x)dx \approx h\left[\frac{f(x_0)}{2} + f(x_1) + f(x_2) + \cdots + f(x_{N-1}) + \frac{f(x_N)}{2}\right]$

3. **适用条件**  
   梯形法则适用于被积函数 $f(x)$ 在区间 $[a, b]$ 上连续的情况。虽然梯形法则的精度不如 Simpson 法则，但它实现简单，适用于快速估算积分值。

#### 梯形法则公式  
梯形法则的公式为：
 $\int_a^b f(x)dx \approx h\left[\frac{f(x_0)}{2} + f(x_1) + \cdots + f(x_{N-1}) + \frac{f(x_N)}{2}\right]$  
其中：
- $x_0 = a, x_N = b$，其余点为 $x_i = a + i \cdot h$。
- $h = \frac{b-a}{N}$ 是步长。

##### 梯形法则的实现简单，但精度较低，尤其在被积函数变化较快时误差较大。

// ... import numpy as np
import matplotlib

matplotlib.use('Agg')  # 使用无图形界面的后端
import matplotlib.pyplot as plt


# 待积分函数
def f(x):
    return x ** 4 - 2 * x + 1


# 梯形法则积分函数
def trapezoidal(f, a, b, N):
    """
    梯形法数值积分
    :param f: 被积函数
    :param a: 积分下限
    :param b: 积分上限
    :param N: 子区间数
    :return: 积分近似值
    """
    h = (b - a) / N
    result = 0.5 * (f(a) + f(b))
    for i in range(1, N):
        result += f(a + i * h)
    return result * h


# Simpson法则积分函数
def simpson(f, a, b, N):
    """
    Simpson法数值积分
    :param f: 被积函数
    :param a: 积分下限
    :param b: 积分上限
    :param N: 子区间数（必须为偶数）
    :return: 积分近似值
    """
    if N % 2 != 0:
        raise ValueError("N 必须为偶数")
    h = (b - a) / N
    result = f(a) + f(b)
    for i in range(1, N, 2):
        result += 4 * f(a + i * h)
    for i in range(2, N - 1, 2):
        result += 2 * f(a + i * h)
    return result * h / 3


def main():
    a, b = 0, 2  # 积分区间
    exact_integral = 4.4  # 精确解

    results = {"N": [], "Trapezoidal": [], "Simpson": [], "Trapezoidal Error": [], "Simpson Error": []}

    for N in [100, 1000]:  # 不同子区间数
        trapezoidal_result = trapezoidal(f, a, b, N)
        simpson_result = simpson(f, a, b, N)

        trapezoidal_error = abs(trapezoidal_result - exact_integral) / exact_integral
        simpson_error = abs(simpson_result - exact_integral) / exact_integral

        results["N"].append(N)
        results["Trapezoidal"].append(trapezoidal_result)
        results["Simpson"].append(simpson_result)
        results["Trapezoidal Error"].append(trapezoidal_error)
        results["Simpson Error"].append(simpson_error)

        print(f"N = {N}")
        print(f"梯形法则结果: {trapezoidal_result:.8f}, 相对误差: {trapezoidal_error:.2e}")
        print(f"Simpson法则结果: {simpson_result:.8f}, 相对误差: {simpson_error:.2e}")
        print("-" * 40)

    # 绘制误差图并保存
    plt.figure()
    plt.loglog(results["N"], results["Trapezoidal Error"], label="Trapezoidal Error", marker='o')
    plt.loglog(results["N"], results["Simpson Error"], label="Simpson Error", marker='s')
    plt.xlabel("Number of Subintervals (N)")
    plt.ylabel("Relative Error")
    plt.title("Error Comparison: Trapezoidal vs Simpson")

### 4.2 误差分析
- 分析两种方法在不同 $N$ 下的误差表现。
- Simpson 法则与梯形法则误差的数量级差异及原因。
- 讨论 $N$ 增大时误差的变化趋势。

#### 1. 两种方法在不同 $N$ 下的误差表现
- **梯形法则**：  
  梯形法则的误差相对较大，尤其在 $N$ 较小时，误差的数量级通常为 $10^{-4}$ 或更高。随着 $N$ 增大，误差逐渐减小，但收敛速度较慢。
  
- **Simpson 法则**：  
  Simpson 法则的误差远小于梯形法则，通常在 $N$ 较小时误差的数量级已经达到 $10^{-8}$ 或更低。随着 $N$ 增大，误差进一步减小，表现出更高的精度。

#### 2. Simpson 法则与梯形法则误差的数量级差异及原因
- **误差来源**：  
  梯形法则使用线性函数（直线）近似被积函数，因此误差与函数的二阶导数相关，误差的数量级为 $O(h^2)$，即步长 $h$ 的平方。  
  Simpson 法则使用二次多项式（抛物线）近似被积函数，因此误差与函数的四阶导数相关，误差的数量级为 $O(h^4)$，即步长 $h$ 的四次方。

- **数量级差异**：  
  Simpson 法则的误差比梯形法则低两个数量级（$O(h^4)$ 对比 $O(h^2)$），这使得 Simpson 法则在相同 $N$ 下表现出更高的精度。

#### 3. $N$ 增大时误差的变化趋势
- **梯形法则**：  
  随着 $N$ 增大，步长 $h = \frac{b-a}{N}$ 减小，误差以 $O(h^2)$ 的速度减小。虽然误差逐渐减小，但收敛速度较慢，需要较大的 $N$ 才能达到较高的精度。

- **Simpson 法则**：  
  随着 $N$ 增大，步长 $h$ 减小，误差以 $O(h^4)$ 的速度减小。由于误差收敛速度快，即使 $N$ 较小，Simpson 法则也能达到很高的精度。

#### 4. 总结
- Simpson 法则在误差表现上明显优于梯形法则，尤其在 $N$ 较小时，误差的数量级差异非常显著。
- 随着 $N$ 增大，两种方法的误差均减小，但 Simpson 法则的收敛速度更快，适合需要高精度的数值积分场景。

## 五、结论
- 总结两种方法的适用场景和优劣。
- 结合本实验结果，说明实际工程中如何选择数值积分方法。


#### 1. 两种方法的适用场景和优劣
- **梯形法则**：
  - **优点**：实现简单，计算量较小，适合快速估算积分值。
  - **缺点**：精度较低，误差收敛速度慢（误差为 $O(h^2)$），在被积函数变化较快时误差较大。
  - **适用场景**：适合对函数变化较平缓、对精度要求不高的场景。

- **Simpson 法则**：
  - **优点**：精度高，误差收敛速度快（误差为 $O(h^4)$），在函数为二次多项式或接近二次多项式时能给出精确结果。
  - **缺点**：实现稍复杂，计算量略大，要求子区间数 $N$ 必须为偶数。
  - **适用场景**：适合对函数变化较快、对精度要求较高的场景。

#### 2. 实验结果分析
- 从实验结果可以看出，Simpson 法则的误差远小于梯形法则，尤其在 $N$ 较小时，误差的数量级差异非常显著。
- 随着 $N$ 增大，两种方法的误差均减小，但 Simpson 法则的收敛速度更快，表现出更高的精度。

#### 3. 实际工程中的选择
- **梯形法则**：适用于对计算速度要求较高、对精度要求不高的场景，例如实时性要求较高的嵌入式系统。
- **Simpson 法则**：适用于对精度要求较高的场景，例如科学计算、工程仿真等需要高精度积分的场景。

#### 4. 总结
在实际工程中，数值积分方法的选择应根据具体需求权衡精度和计算效率。如果函数较复杂且精度要求高，优先选择 Simpson 法则；如果函数较简单且对精度要求不高，可以选择梯形法则以提高计算效率。
